Add multiple admin emails for notification such as new user registration.

== Description ==
	
The Plugin helps to add multiple admin emails for notification such as new user registration. The plugin does not use database so that the execution will be quite faster.

*   Compatible with IE, Firefox, Safari, Opera, Chrome
*   With Free version you can add only one extra admin email, which means your default admin email and one extra admin email.
*   With Pro version you can add as much as extra admin emails.

Checkout Pro Version screenshots
[Pro Version](http://www.bilogicit.com/multi-admin-emails/)
